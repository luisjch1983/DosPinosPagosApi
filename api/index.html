TOWN-COUNTRY-WEATHER-CURRENCY API

This API is an example of a few Microservices to retrieve Town, Country, Weather and Currency data.

The API is written in NodeJS and the results are returned in JSON format.

FUNCTIONS OF THE API

Function URL Example
/countries List of all countries
/countryid2/:id2 List country data with country code (2 chars like "DE")
/countryname/:name List country by english name (like "Germany")
/towns List of all towns >15K population or a capital towns
/townname/:name List town by name (like "Munich")
/townid/:id List town by id
/towncc/:cc List all towns by country code (2 chars like "DE")
/weather/:town List weather of a town by name(like "Munich")
/weathercondition/:town List weather condition of a town by name(like "Munich")
/currencies List of all currencies and their values compared to CZK (Czech Crown)
/currency/:curr Value of a currency (like "EUR") compared to CZK (Czech Crown)
